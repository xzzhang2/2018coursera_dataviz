This page is for visualizing network data in d3. The codes are from: https://bl.ocks.org/martinjc/871febe9277e731479384e39f84f1cf8 and http://bl.ocks.org/emeeks/9441864. The song cover data is collected by Zhang and Zhang (2017).

This graph indicates the song cover relationship among the contestants and original singers. For example, contestant A sings a song from Beatles, then there will be a link from A to Beatles. Obviously, this graph needs to improve because the node labels are not avaiable. But nevertheless, from eyeballing the graph one can clearly observe a "central-perriphery" tendency between some famous singers (clustering in the central) and some "alternative pairs" (drfting away in the margins).
